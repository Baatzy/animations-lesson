CSS Transition

Transitions cause changes to a property to take place over a period of time (duration).

Using the transition property we can define which CSS property to transition, how long it takes etc. The broswer will interpolate values for the property over the time.

transition: property(defaults to All) duration(needs to be specified) timing-function(defaults to ease) delay(defaults to 0)
  • Order of the transition parameters is irrelevant so long as you have a duration specified. (duration must be specified before delay)
  • Not all CSS properties can be transitioned, only those with middle transition states
  • Properties that CAN be transitioned:
    • background-color
    • color
    • opacity
    • font-size
    • padding
    • margin
    • box-shadow
    • ...
  • Properties that CANNOT be transitioned:
    • visibility
    • font-weight
    • font-family
    • ...
Example: 
transition: opacity 0.25s;

Note about Browser Support

Until all browsers adopt the standard version, you will want to include vendor prefixes to allow Cross-Browser Compatibility. This holds true for the transition property as well as other animation based css properties described in this article.

Here is an example of vendor prefixes with the transition property:

-webkit-transition: background-color 3s;
-moz-transition: background-color 3s;
-o-transition: background-color 3s;
-ms-transition: background-color 3s;
transition: background-color 3s;
Rather then type all these out by hand, you might consider using a build tool like Webpack or PostCSS

CSS Transform

The transform property lets you modify elements in their coordinate space by skewing, rotating, translating or scaling. It can be incredily useful when combined with the transition property to create rich animations

transform: function(params);

transform functions:

  • skew(degrees)
  • rotate(degrees)
  • translate(x, y)
  • scale(number)
  • matix(number, ...)

Skew

Skews an element along the X and Y-axis by the given angle

Skew Me! 
transform: skewX(20deg);

Rotate

Rotates an element clockwise or counter-clockwise according to a given degree

Rotate Me! 
transform: rotate(360deg);

Translate

Moves an element from its current position

Translate Me! 
transform: translate(15px, -15px);

Scale

Increases or decreases the size of an element

Scale Me! 
transform: scale(1.5);

Keyframe Animations

What is a Keyframe Animation?

A keyframe animation is a detailed list of how css property animation takes place over time.

There are two key parts to creating a keyframe animation:

  1. Keyframe Definition - defines the stages and styles of the animation using @keyframes
  2. Animation Properties - assign @keyframes to specific element and specify animation details

#1 Keyframes

Each @keyframes is composed of:

  • Name of the animation: A name that describes the animation. e.i. rainbow
  • Steps of the animation: Each step is represented as a percentage from (initial step) 0% to (final step) 100%.
  • CSS Properties
@keyframes animation-name {
  step1 (to, from, 0%) {
    property: value;
  }
  step2 (33%, 50%, ...) {
    property: value;
  }
}

@keyframes rainbow {
  0%, 100% {
    color: red;
  }
  33% {
    color: green;
  }
  66% {
    color: blue;
  }
}

#2 Animation Properties

Animation properties do 2 things:

  1. Assign the @keyframes to the elements that you want to animate
  2. Define how the animated plays

The first 2 parameters are requred

  • animation-name (required)
  • animation-duration (required)
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-fill-mode
  • animation-play-state

You can define each of the above properties indevidually, or use the shorthand:

.target {
  animation: rainbow 1s;
}

Color Keyframes

Rainbow 
@keyframes rainbow {
  0%, 100%{
    color: red;
    border-color: red;
  }
  33% {
    color: green;
    border-color: green;
  }
  66% {
    color: blue;
    border-color: blue;
  }
}

.rainbow {
  animation: rainbow 2s infinite;
}

Transform Keyframes

Shake 
@keyframes shake {
  0%, 100% {
    transform: translate3d(0, 0, 0);
  }

  10%, 30%, 50%, 70%, 90% {
    transform: translate3d(-10px, 0, 0);
  }

  20%, 40%, 60%, 80% {
    transform: translate3d(10px, 0, 0);
  }
}

.shake:hover {
  animation: shake 1s;
}

What is Anime.js?

A "javascript animation engine"

animejs.com

Transition Input Value

Count 
var numberCounter = anime({
  targets: '#numberCounter',
  value: 1000,
  round: 1,
  easing: 'easeInOutExpo'
});

  targets: '#cube',
  translateX: 250,
  direction: 'alternate'
